# AI-Powered-Real-Time-ECG-Signal-Diagnostic-System-Using-Signal-Processing-and-Deep-Learning
The Electrocardiogram (ECG) signal classification is a critical component in the early detection and diagnosis of cardiac arrhythmias. However, manual clinical analysis of continuous ECG recordings is time-consuming and prone to human error. While machine learning offers a viable pathway for automation, deep learning models often suffer from severe performance bias due to the inherent class imbalance in medical datasets, where normal heartbeats heavily outnumber abnormal anomalies. This project details the iterative development, optimization, and scaling of a robust, automated ECG classification framework utilizing a 1-Dimensional Convolutional Neural Network (1D-CNN).
The development methodology progressed through three distinct phases: signal preprocessing, architectural optimization, and large-scale validation. Initially, raw ECG signals were purified using a zero-phase Butterworth bandpass filter 0.5 Hz to 40 Hz to eliminate baseline wander and high-frequency environmental noise before segmenting the continuous stream into fixed 200-sample heartbeat windows. To address early model bias—where initial training yielded high overall accuracy but a dangerously high rate of false negatives for cardiac distress—we implemented a multi-tiered mitigation strategy combining minority class upsampling and algorithmic class-weighting. The core model architecture was subsequently refined through automated hyperparameter tuning using Keras Tuner, optimizing filter counts, kernel sizes, and learning rates.
